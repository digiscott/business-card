# Out of Scope for v1

Do not add these items in the first version.

## Content exclusions

- Case studies
- Work samples
- Project artifact galleries
- Detailed employer timeline
- Detailed employment history
- Full web resume
- Blog posts
- Testimonials or references section
- Personal family details
- Confidential employer details
- Severance or layoff details
- Internal company artifacts

## Feature exclusions

- Contact form
- Calendar scheduler
- vCard download
- Newsletter signup
- CMS
- Database
- Authentication
- User accounts
- Comments
- Search
- Embedded chatbot iframe
- Analytics by default
- Tracking-heavy scripts
- Server-side personalization

## Technical exclusions

- API routes
- Server actions
- Server-side rendering
- Runtime server dependency
- Image optimization requiring a Next.js server
- Database integration
- Middleware requiring server runtime
- Heavy animation libraries
- Complex UI frameworks beyond what is needed

## Design exclusions

- Overly corporate styling
- Resume-like dense page layout
- Carousel-dependent content
- Text embedded in images
- Hover-only interactions
- Low-contrast color palettes
- Motion required to understand content

## Revisit later

These could be added in a later version after v1 is live:

- Case-study links
- Selected work artifacts
- Branded OG image
- Simple analytics
- Contact form using a third-party static form service
- Calendar scheduling link
- More polished illustration system
- Downloadable vCard
- Dedicated resume page
- More robust chatbot source transparency page
